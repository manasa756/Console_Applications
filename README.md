#  ATM Machine Simulation - C# Console Application

##  Project Overview
ATM Machine Simulation is a menu-driven console application built using C#.  
The project simulates real-world ATM operations such as secure PIN login, balance checking, deposits, and withdrawals.  
This project was created to practice Object-Oriented Programming concepts and exception handling in .NET.
