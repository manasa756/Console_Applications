#  Task Manager Console Application 

##  Project Overview

This is a **Task Manager Console Application** developed using **C#**.
The project allows users to manage daily tasks through a simple console interface. It demonstrates core programming concepts like **Collections, Classes, Methods, and OOP principles**.

---

##  Features

✔️ Add New Tasks
✔️ View All Tasks
✔️ Delete/Remove Tasks
✔️ Mark Tasks as Completed

---

## Technologies Used

* C#
* .NET Console Application
* Collections (List)
* Object-Oriented Programming (OOPS)

---

## Concepts Practiced

* Classes & Objects
* Properties & Methods
* Lists and Data Handling
* Conditional Statements
* Loops


---

##  How to Run the Project

1. Open the project in **Visual Studio**.
2. Build the solution.
3. Run the application using: Ctrl + F5
4. Follow the menu options displayed in the console.
